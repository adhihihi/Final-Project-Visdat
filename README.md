# World Happiness Dashboard

> Dashboard interaktif yang memvisualisasikan skor kebahagiaan global (Life Ladder) dari 138 negara (2005–2023) menggunakan data nyata dari World Happiness Report (Gallup World Poll).

Demo: https://[nama-kelompok].vercel.app

## Isi Dashboard
- Chart 1: Bar Chart — Top 15 Negara Paling Bahagia (2023)
- Chart 2: Line Chart — Tren Kebahagiaan dari Tahun ke Tahun (Indonesia, Finland, Singapore, World Average)
- Chart 3: Scatter Plot — Hubungan antara Log GDP per Kapita dengan Skor Kebahagiaan (2023)
- Chart 4: Doughnut Chart — Rata-rata Skor Kebahagiaan per Wilayah/Region Dunia (2023)
- Fitur interaktif: Tooltip interaktif pada hover, toggle visibilitas data negara via legenda Line Chart, dan dropdown filter untuk menyorot (highlight) negara tertentu secara dinamis pada Scatter Plot beserta ringkasan metriknya.
- Animasi: Animasi entrance bawaan Chart.js, Count-up angka KPI (Jumlah Negara & Peringkat Indonesia), dan animasi CSS fade-in bertahap (staggered) pada elemen kartu saat halaman pertama kali dimuat.

## Sumber Data
- Nama dataset: World Happiness Report 2024 · Gallup World Poll
- URL sumber: https://worldhappiness.report/data-sharing/

## Cara Jalankan di Lokal

# Jalur A (static):
Buka index.html langsung di browser (atau pakai Live Server di VS Code)

# Jalur B (server):
npm install
npm start

# Buka http://localhost:3000

## Teknologi
- Chart.js (visualisasi)
- HTML + CSS + JavaScript
- Vercel (deployment)
